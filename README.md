# Spring + Hibernate Project

Проект для демонстрации интеграции Spring Framework и Hibernate ORM.

## Функциональность
- Настройка Spring + Hibernate
- Сущности User и Car с связью One-to-One
- DAO и Service слои
- HQL запросы для поиска данных

## Технологии
- Java
- Spring Framework
- Hibernate ORM
- MySQL
- Maven
- Lombok
